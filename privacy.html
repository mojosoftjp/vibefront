プライバシーポリシーPrivacy Policy

最終更新: 2026年3月12日Last updated: March 12, 2026

VibeFront（以下「本アプリ」）は、Mojosoft が開発・提供するWindows向けHTMLエディタアプリです。本アプリは、お客様のプライバシーを尊重し、個人情報の保護に努めています。 VibeFront ("the App") is a Windows HTML editor application developed by Mojosoft. We respect your privacy and are committed to protecting your personal information.

収集する情報Information We Collect

本アプリは個人情報を収集しません。 The App does not collect any personal information.

  • ユーザー登録やアカウント作成は不要です。No user registration or account creation is required.
  • 利用状況やアクセス解析データを外部サーバーに送信することはありません。No usage data or analytics are sent to external servers.
  • 編集中のファイル内容は、すべてお客様のPC上でのみ処理されます。All file content being edited is processed entirely on your local PC.

AI機能についてAI Features

本アプリのAI機能を使用する場合、お客様が入力したテキストや画像は、お客様自身が設定したAIプロバイダー（OpenAI、Anthropic、Google Gemini、またはローカルLLM）のAPIに送信されます。これらのデータの取り扱いについては、各AIプロバイダーのプライバシーポリシーをご確認ください。 When using AI features, the text and images you provide are sent to the AI provider API that you have configured (OpenAI, Anthropic, Google Gemini, or a local LLM). Please review each AI provider's privacy policy regarding how they handle this data.

ローカルLLM（LM Studio、Ollama等）を使用する場合、データはネットワーク外に送信されません。 When using local LLMs (LM Studio, Ollama, etc.), your data never leaves your network.

ローカルに保存されるデータLocally Stored Data

本アプリは、以下のデータをお客様のPC上（%LOCALAPPDATA%\VibeFront\）にのみ保存します： The App stores the following data only on your local PC (under %LOCALAPPDATA%\VibeFront\):

  • ウィンドウ位置・サイズなどのUI設定Window position, size, and UI preferences
  • AI プロバイダーのAPIキー設定AI provider API key configuration
  • 最近使ったファイルの一覧Recent file list
  • パネルのドッキングレイアウトPanel docking layout
  • 自動保存用の一時ファイルAuto-save temporary files
